rh.core

Core RH (Reactive Html Framework) module.

This module implements the main MeshBuilder class for creating reactive computational meshes with interactive web interfaces.

class rh.core.MeshBuilder(mesh_spec: ~typing.Dict[str, ~typing.Any], functions_spec: ~typing.Dict[str, ~typing.Any], initial_values: ~typing.Dict[str, ~typing.Any] = <factory>, field_overrides: ~typing.Dict[str, ~typing.Any] = <factory>, ui_config: ~typing.Dict[str, ~typing.Any] = <factory>, output_dir: str | None = None)[source]

Main class for building reactive mesh applications.

A mesh defines relationships between variables through functions, enabling bidirectional dependencies and real-time updates in web interfaces.
